# AI搭子 - 内容创作者AI协作助手 - 产品规划

## 产品定位
一句话描述：为内容创作者提供 AI 协作助手，覆盖从灵感到发布的全流程——AI辅助写作、智能排版、多平台发布建议、数据复盘

## 目标用户
- 主要用户群体：独立内容创作者、自媒体人、公众号/小红书/抖音博主
- 用户痛点：写作卡壳缺灵感、排版费时、多平台发布重复劳动、不知道内容表现如何
- 使用场景：日常内容创作、选题规划、多平台分发、内容复盘优化

## 核心功能
按优先级排列，最多 5 个核心功能：
1. AI辅助写作（续写/改写/扩写/缩写/标题生成）（必须有）
2. 智能排版预览（公众号/小红书/抖音三种格式实时预览）（必须有）
3. 内容日历+选题库（选题收藏、发布时间线、草稿管理）（必须有）
4. AI内容复盘分析（标题评分、关键词建议、发布时间优化）（可选）
5. 一键多平台格式导出（适配各平台字数/格式要求）（可选）

## 变现设计
- 变现方式：Freemium + 订阅制
- 免费功能：基础AI写作（3次/天）、单平台排版、5个草稿
- 付费功能：无限AI写作、多平台排版、内容日历、数据分析、团队协作
- 定价策略：免费版够用，Pro版 29元/月，Creator版 59元/月

## 技术方案
- 目标平台：Web
- 前端框架：纯 HTML + CSS + Vanilla JS（快速开发，无框架依赖）
- 后端方案：纯前端无需后端（localStorage存储）
- 数据存储：localStorage（草稿/选题/设置）
- 第三方依赖：无（AI功能预留接口，演示模式用模板匹配）
- 部署方式：GitHub Pages
